Hospital Performance Analysis Dashboard (Power BI)
📌 Project Overview

The Hospital Performance Analysis Dashboard is an interactive Business Intelligence solution built using Power BI to analyze and visualize hospital operations, clinical performance, and financial metrics.
The dashboard transforms raw hospital data into actionable insights to support data-driven decision-making for administrators and healthcare stakeholders.

🎯 Objectives

Analyze patient admissions, demographics, and discharge trends

Monitor doctor workload and department-wise performance

Evaluate bed occupancy, medical tests, surgeries, and staff utilization

Track hospital billing, revenue, and cost distribution

Enable interactive exploration using slicers, filters, and drill-downs

Demonstrate the effectiveness of Power BI in healthcare analytics

🗂️ Dataset Description

The project integrates data from multiple hospital-related sources, including:

Patients & Appointments

Doctors & Departments

Beds & Rooms

Medical Tests & Surgeries

Billing & Finance

Medical Stock & Medicines

Staff Information

Patient Satisfaction Scores

🧱 Data Model

Implemented using a Star Schema Architecture

Fact Tables: Appointments, Bills, Patient_Tests, Medicine_Patient

Dimension Tables: Patients, Doctors, Departments, Staff, Beds, Calendar

Includes What-If parameter tables for scenario analysis

Optimized for performance, scalability, and accurate DAX calculations

🧮 Data Preparation & DAX

Data cleaning and transformation using Power Query

Created Calculated Columns, Measures, and Tables

Implemented:

Time intelligence using a Calendar table

Patient age categorization

Status icons for appointments, tests, and surgeries

Financial KPIs (revenue, salary, fees, costs)

What-If analysis using GENERATESERIES()

📊 Dashboard Pages

Home Page – Navigation and overview

Overview Dashboard – High-level KPIs

Patient Dashboard – Admissions, demographics, discharge trends

Doctor Dashboard – Workload and appointment analysis

Hospital Dashboard – Beds, tests, surgeries, staff

Finance Dashboard – Billing, revenue, costs

📈 Key Visualizations

KPI Cards

Bar & Column Charts

Line & Area Charts

Donut Charts

Stacked Bar Charts

Tables & Matrix Visuals

Interactive Slicers (Date, Doctor, Patient, Department)

🛠️ Tools & Technologies

Microsoft Power BI

Power Query (ETL)

DAX (Data Analysis Expressions)

Excel (Data Source)

Star Schema Data Modeling

🚧 Limitations

Uses historical data only

No real-time data integration

Role-Level Security (RLS) not implemented

Predictive analytics limited to basic forecasting

Manual data refresh required
